# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档提供详细说明
- 导入调试视频、讲解视频及开发工具安装包
- 系统分为管理员、家政公司、家政阿姨和普通用户四个角色登录

## 适用场景
- 家政保洁服务公司
- 个人或家庭用户进行家政保洁服务预约

## 技术特点
- 采用B/S架构、MVC模式，基于Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
